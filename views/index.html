US States REST API - Jordan Huynh

Hello! Use /states to access the API.

Example endpoints:

  • /states/ - all states
  • /states/KS - single state
  • /states/KS/funfact - random fun fact
  • /states/KS/capital - capital city
  • /states/KS/nickname - nickname
  • /states/KS/population - population
  • /states/KS/admission - admission date
